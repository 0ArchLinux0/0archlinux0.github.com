```java
class Solution {
    public int removeDuplicates(int[] nums) {
      int newIdx = 0, curIdx = 1;
      while(curIdx < nums.length) {
        if(nums[curIdx] != nums[newIdx]) nums[++newIdx] = nums[curIdx];
        curIdx++;
      }
      return newIdx + 1;
    }
}
```
